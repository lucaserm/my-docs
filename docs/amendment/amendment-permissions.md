# Permissões da Emenda

---

### `Proposicao.NovaMovimentacao.Emenda.Editar`

- Cria Emenda
- Edita Emenda
- Lê anexos
- Insere anexos
- Insere signatários
- Envia para protocolar

### `Proposicao.NovaMovimentacao.Emenda.Excluir`

- Exclui Emenda

### `Proposicao.NovaMovimentacao.Emenda.ImprimirRecibo`

- Imprime Recibo de Emenda
